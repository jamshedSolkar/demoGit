Git Repo practice

git clone <here git url> (This is will create desktop clone in your local machine)
git branch (This will shows you list of branch Name)
git checkout -b "BranchName" (This will shows you list of repo)
git add -A (This will add all your files stored in your project)
git commit -m "write your commit here" (you have to commit your code before push/pull, so that other developer knows the changes)
git push origin "currentBranchName" (this CMD line will bring your files/folder in staging level)
git pull origin "currentBranchName" (this CMD line will update your files/folder in your branch, so that other developer can view your updates)

Extra git CMD

git status (this will show your status of your files)
git log (this will show all your log of add and edit files/folder)
git rm (this will remove your file from your branch)
